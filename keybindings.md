# VIM Plugings

## VIM and R

- `\rf`     Open the terminal buffer with R
- `\rq`     Close the R session
- `\aa`     Send entire file
- `\bb`     Send block
- `\ff`     Send function
- `\ss`     Send Selection
- `\l`      Send line
- `\ro`     Object explorer
- `\rl`     View this object
- `\rh`     Help
- `\re`     Different help
- `\rv`     View dataframe
- `\o`      Output of current line is rendered as a comment 
- `__`      Change to `->`
- zM        Close all folds
